# Task: Outlook API

Pre-requisite(s)

NodeJS
NPM or Yarn
To Run.

In the project directory, you can run:

git clone https://github.com/pezemiwe/Outlook_API_Task.git

## Installation

```bash
npm install
```

```bash
npm start
```



Installs and runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
